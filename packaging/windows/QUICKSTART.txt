ProtoSwitch v0.1.0-beta.11

Быстрый старт

1. Если это installer-версия, первичная инициализация уже выполнена.
2. Если это portable-версия, запустите:
   protoswitch.exe init --non-interactive --no-autostart
3. Проверьте состояние:
   protoswitch.exe doctor
4. Для фоновой работы включите автозапуск:
   protoswitch.exe autostart install
5. Для ручной смены proxy:
   protoswitch.exe switch
6. Запуск protoswitch.exe без аргументов открывает консоль управления.
